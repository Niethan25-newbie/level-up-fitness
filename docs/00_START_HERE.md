### Step 1: Import Database
```sql
Open phpMyAdmin or MySQL client
Copy content of: /sql/database.sql
Execute the SQL script
```

### Step 2: Configure Database
Edit `/config/database.php`:
```php
define('DB_USER', 'root');
define('DB_PASS', 'your_mysql_password');  // CHANGE THIS
```

### Step 3: Configure App URL
Edit `/config/config.php`:
```php
define('APP_URL', 'http://your-domain.com/level-up-fitness/');
```

### Step 4: Upload to Server
- Copy all files to your web server
- Ensure proper permissions (755 for files, 777 for uploads if needed)
- Place in your web root directory
